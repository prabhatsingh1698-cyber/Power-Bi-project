# Power-Bi-project
This Is  IPL Data Analysis using (Power BI)
Built an end-to-end interactive dashboard to analyze IPL match and player data.
– Performed data transformation and cleaning using Power Query.
– Applied DAX functions to calculate runs, strike rate, win rate and wickets.
– Designed dynamic dashboards with slicers and filters for deep analysis.
– Generated insights on team-wise, player-wise, and season-wise performance trends
