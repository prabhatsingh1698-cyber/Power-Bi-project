🔹 Project Title: IPL Data Analysis Dashboard (Power BI)
📌 Project Description
Developed an interactive Power BI dashboard to analyze IPL (Indian Premier League) data.
Transformed raw cricket datasets into meaningful insights using data visualization techniques.
📊 Key Features
Created dynamic dashboards to track team performance, player statistics, and match outcomes.
Designed KPIs such as total runs, strike rate, win percentage, and top performers.
Implemented filters and slicers for season-wise, team-wise, and player-wise analysis.
Visualized trends like highest run scorers, leading wicket-takers, and winning teams.

🛠️ Tools & Technologies Used
Power BI (Data Visualization & Dashboarding)
Microsoft Excel / CSV (Data Source)
Power Query (Data Cleaning & Transformation)
DAX (Data Analysis Expressions for calculations)

📈 Insights Generated
Identified top-performing teams and players across seasons.
Analyzed match-winning factors such as toss impact and venue influence.
Highlighted consistency of players using performance metrics.
Compared team performance across different IPL seasons.

🔄 Data Processing
Cleaned and transformed raw IPL datasets using Power Query.
Handled missing values and ensured data consistency.
Created calculated columns and measures using DAX.

🎯 Outcome
Delivered a user-friendly and interactive dashboard for cricket analytics.
Improved data storytelling and decision-making using visual insights.
